# slack-to-scrapbox

個人SlackでWebの記事に対してコメントや感想を残していたメッセージをScrapboxに移すために書いたスクリプトです

## こだわり
- スレッドの内容もまとめる扱える
- attachmentsを利用してサムネイルを設定する

## ざっくりとした使い方
- slackのスタンダードエクスポートを使ってjsonファイルを手に入れる
- `config.sample.ts`をもとに`config.ts`を用意する
- コードをいじって自分好みのページになるようにする
- `npm start`
- Googleログインをする
  - やりたければここも自動化できます
- あとはひたすら待つ
